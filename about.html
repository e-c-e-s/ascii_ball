What is ASCII PONG?

ASCII PONG remake of an iconic arcade game Atari PONG which came in 1972. While there are thouthands of such clones, nearly all of them have some sort of GUI. ASCII PONG, as the name suggests, relies only on Text User Interface for displaying the game and runs completely in command prompt! It is rather a proof-of-concept (a fast-paced video game can be done without any knowledge about GUI programming or computer graphics), but is still playable and hopefuly enjoyable ^_^
Platform: Windows
No. of players: 1-2
Created by: souserge
